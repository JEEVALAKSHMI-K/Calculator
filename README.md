# 🧮 Java Console Calculator

A simple, interactive command-line calculator built using Java.

## ✨ Features
* Performs Basic Arithmetic: `+`, `-`, `*`, `/`
* Safe Division: Prevents crashes when dividing by zero.
* Continuous Loop: Run multiple calculations without restarting.

## 🚀 How to Run
1. Compile the program:
```bash
   javac Calculator.java
